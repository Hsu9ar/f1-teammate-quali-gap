# F1 チームメイト予選差

[English](./README.md) | 日本語

F1 のあるシーズンを通して、2 人のチームメイトの**単独走行(予選)ペース**を**予選差の中央値**で比較する、単一ファイルのインタラクティブ Web アプリです。中央値はクラッシュ・黄旗・マシントラブルといった外れ値に強い指標です。

🔗 **デモ:** _(GitHub Pages を有効化し、URL をここに記載。例:`https://<ユーザー名>.github.io/<リポジトリ>/f1-teammate-quali-gap.html`)_

## なぜ平均ではなく中央値か

予選差には「汚染されたサンプル」が多く含まれます。クラッシュ、黄旗で抹消されたラップ、Q1 敗退で 1 本しか走れなかったケースなど。これらは極端な値を生み、**平均**を実態から引き離します。**中央値**は外れ値を無視するため、2 人の典型的な差をはるかに正確に表します。

## 指標の計算方法

各レースについて:

1. 両ドライバーが到達した**最も上位の共通予選セッション**(Q3 > Q2 > Q1)の最速ラップを取得 — 常に対等な比較にするため。
2. コース長の違いを排除するため、生のタイムではなく**百分率**で比較:
   `gap% = (t_A − t_B) / t_B × 100`
3. シーズンを通したその%差の**中央値**をとる。

注意:データ源はドライ/ウェットを区別しません。ウェットのレースは手動で除外してください(バーまたは表の行をクリック)。本指標は予選/単独走行のペースのみを表し、決勝ペースは反映しません。

## 機能

- **シーズン → コンストラクター → 2 ドライバー**を選択。すべて即時に再計算。
- **中央値 / 平均 / 対戦成績 / サンプル数**のサマリーカード。
- レースごとの棒グラフ。**極端な外れ値をクリップ**する堅牢な縦軸(実際の値はバー上に表示)で、通常の差が読みやすいまま。チェックを外すと全範囲表示。
- **任意のバー/表の行をクリックして除外**。中央値はリアルタイムで再計算。
- UI は **English / 日本語 / 中文**(右上で切替)。
- **ローカル優先データ**:オフラインや、ライブ API が到達不可な地域でも動作。

## データ源

無料の [Jolpica-F1 API](https://github.com/jolpica/jolpica-f1)(廃止された Ergast API の後継)。API キー不要。

## 実行方法

単一の静的 HTML ファイルです。`f1-teammate-quali-gap.html` をブラウザで開くだけ。

ページはまずローカルデータ(`data/quali-<year>.json`)を読み込み、なければライブ API にフォールバックします。ライブ API に接続できない場合(ネットワーク / CORS / 地域制限)、一度だけローカルデータを生成してください:

```bash
python build_data.py 2024            # 1 シーズン
python build_data.py 2021 2022 2023 2024   # 複数
```

`data/quali-<year>.json` が生成されます。ページを再読み込みすればオフラインで動作します。

### オプション:FastF1 でより豊富なデータ

`build_data.py` には `build_with_fastf1()` の例が含まれています。[FastF1](https://docs.fastf1.dev/) は Python ライブラリ(ブラウザ用 API ではありません)で、テレメトリ・タイヤ・フリー走行データを取得できます。`pip install fastf1` でインストールし、`main()` 内の呼び出しを切り替えてください。

## GitHub Pages へのデプロイ

1. リポジトリを作成し、これらのファイルを push(下記参照)。
2. GitHub で **Settings → Pages → Build and deployment → Source: Deploy from a branch**、ブランチ `main`、フォルダ `/ (root)`、**Save**。
3. 1 分ほどで `https://<ユーザー名>.github.io/<リポジトリ>/f1-teammate-quali-gap.html` で公開されます。

## プロジェクト構成

```
f1-teammate-quali-gap.html   # アプリ本体(HTML + CSS + JS、単一ファイル)
build_data.py                # Jolpica → data/quali-<year>.json(FastF1 任意)

README.md / README.ja.md     # ドキュメント
```
