# F1 チームメイト予選差

[English](./README.md) | 日本語 | [中文](./README.zh.md)

F1 のチームメイト同士の予選ペースを比較するためのインタラクティブなブラウザツールです。**予選差の中央値**を使うことで、クラッシュ、黄旗、機械トラブル、ウェット、明らかに妨げられたラップなどの外れ値の影響を抑えます。

## このツールでできること

- シーズンごとの F1 予選結果を読み込む。
- コンストラクターを選び、任意の 2 人のチームメイトを比較する。
- レースごとの%差、シーズン中央値、平均値、対戦成績、有効サンプル数を計算する。
- レースごとの差を棒グラフで可視化し、極端な外れ値にはフォーカス表示を使う。
- グラフのバーまたは表の行をクリックして、代表性の低いレースを除外する。
- 英語、日本語、中国語の UI に対応する。

## 使い方

1. `f1-teammate-quali-gap.html` をブラウザで開くか、静的ページとしてホストします。
2. シーズン、コンストラクター、ドライバー A、ドライバー B を選択します。
3. 上部カードで中央値、平均値、対戦成績、有効サンプル数を確認します。
4. 棒グラフでレースごとの差のばらつきを確認します。
5. ウェット、クラッシュ、機械トラブル、代表性の低いラップがある場合は、該当するバーまたは表の行をクリックして除外します。
6. **フォーカス表示**で、外れ値をクリップした読みやすい表示と全範囲表示を切り替えます。

## 指標の読み方

差はドライバー A のラップタイムをドライバー B に対する%で表します。

```text
gap% = (t_A - t_B) / t_B * 100
```

- 正の値:ドライバー A がドライバー B より遅い。
- 負の値:ドライバー A がドライバー B より速い。
- 中央値:有効なレースごとの差を並べたときの典型的なシーズン差。
- 平均値:参考にはなるが、極端なサンプルの影響を受けやすい。

## 計算方法

各予選について:

1. 両ドライバーが到達した最も上位の共通セッションを探します。優先順は Q3、Q2、Q1 です。
2. その共通セッションでの各ドライバーの最速ラップを取得します。
3. ラップタイム差を%に変換します。
4. 除外されていない全レースの%差からシーズン中央値を計算します。

これにより Q3 ラップと Q1 ラップのような不公平な比較を避け、サーキット長の違いによる秒差の影響も抑えられます。

## データ源

無料の [Jolpica-F1 API](https://github.com/jolpica/jolpica-f1) を使用します。これは Ergast API のコミュニティ後継で、API キーは不要です。

ページはまずローカルデータを読み込みます。

```text
data/quali-<year>.json
```

ローカルデータがない場合は、ライブ API にフォールバックします。

## ローカルデータの生成

ローカル JSON を用意すると読み込みが速くなり、ライブ API に接続できない環境でも使えます。

```bash
python build_data.py 2024
python build_data.py 2021 2022 2023 2024
```

スクリプトは `data/quali-2024.json` のようなファイルを生成します。

## FastF1 による拡張

`build_data.py` には `build_with_fastf1()` の例があります。[FastF1](https://docs.fastf1.dev/) はブラウザ API ではなく Python ライブラリですが、テレメトリ、タイヤ、フリー走行データを追加する拡張に利用できます。

## 制限事項

- データ源はドライとウェットを区別しません。
- 抹消ラップ、トラフィック、黄旗、機械トラブルなどは手動で判断して除外する必要があります。
- この指標は予選と単独走行ペースのみを表し、決勝ペース、タイヤ管理、戦略遂行力は測りません。

## プロジェクト構成

```text
f1-teammate-quali-gap.html   # 静的 Web アプリ
build_data.py                # データ出力スクリプト
data/                        # 任意のローカル JSON データ
README.md                    # 英語ドキュメント
README.ja.md                 # 日本語ドキュメント
README.zh.md                 # 中国語ドキュメント
```
